# ATUDGUM
这是ATUDGUM的相关代码和数据。数据在main分支中。

## StaticAnalysisTool
该项目用于分析导致测试代码过时的生产代码变更类型并构建提示。在StaticAnalysisTool/src/use/Usage.java中有具体示例演示如何使用该项目进行代码变更类型分析和提示构建。

## jars
此文件夹包含一些JAR文件，您需要将其导入到StaticAnalysisTool项目中才能运行它。

## eval
该文件夹包含一些用于评估实验结果的Python程序。
